CPU Information Tool
++++++++++++++++++++++++++++++++++++++++++
In this Tool, reading /proc/cpuinfo file and determining no. of CPU's available.

- To run this tool, clone https://github.com/kpitak/CPUInfo.git 
- Executable is available in bin folder of this project.
- Run executable to use this tool.
- To open project, open QTCreator. You need the latest QT Creator to open project.
- Go to File->"Open File or Project"->Browse to Source code folder and select *.pro file.
